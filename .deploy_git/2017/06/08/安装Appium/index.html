安装Appium

发表于 2017-06-08 | 分类于 appium

1. 安装依赖

操作系统版本 ubuntu 16.04 LTS， 64-bit

1.0. 安装 Node.js

官方网站下载 Node.js 源码
解压文件安装，查看版本

1.1. 安装 Android SDK

Android 开发者网站下载android-studio
解压缩安装包到当前目录

1.2. 安装 JDK

Oracle 官网下载 rpm 格式 JDK

测试是否完成安装

1.3. 配置 JAVA_HOME 和 ANDROID_HOME

查找 Java 安装路径

将以下添加到 /etc/bash.rc
export JAVA_HOME=< Java 安装路径 >
export ANDROID_HOME=< Android SDK 安装路径 >
export PATH=$PATH:$JAVA_HOME:$ANDROID_HOME:$JAVA_HOME/bin

检查是否配置成功
